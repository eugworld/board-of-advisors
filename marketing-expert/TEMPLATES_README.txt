MARKETING EXPERT SKILL - TEMPLATES & RESOURCES
===============================================

This folder contains professional marketing visual templates and AI image generation prompts.

TEMPLATES (HTML Files)
======================
All templates are self-contained HTML files that can be opened in any browser and screenshotted for social media.

1. linkedin-carousel.html
   - LinkedIn carousel slide template
   - 1080x1080px format
   - Features: Slide number indicator, title area, body text, brand bar
   - Usage: Edit the text content and screenshot for carousel posts

2. quote-card.html
   - Quote/insight card for key takeaways
   - 1080x1080px format
   - Features: Large quotation mark, centered quote, attribution line
   - Usage: Copy template, edit quote and attribution, screenshot

3. comparison-post.html
   - Side-by-side comparison (Old Way vs New Way)
   - 1080x1080px format
   - Features: Left/right layout with visual contrast, VS badge in center
   - Usage: Edit bullet points on each side, screenshot

4. how-to-card.html
   - Step-by-step guide template
   - 1080x1350px format (Instagram portrait, works for LinkedIn)
   - Features: Title, 5 numbered steps with descriptions, CTA section
   - Usage: Edit title and step content, screenshot

DESIGN SPECIFICATIONS
====================
- Brand Color: #2ABFAB (mint green/teal)
- Font Stack: System fonts (-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif)
- All colors are CSS variables for easy customization
- Clean, minimal, professional design
- No external dependencies required

CUSTOMIZATION TIPS
===================
1. Open any HTML file in your browser
2. Edit the text content directly in the HTML (look for visible text areas)
3. To change colors, edit the CSS variables at the top of the <style> section
4. Screenshot the browser window for your social media posts
5. All templates are 1080px wide (standard for Instagram/LinkedIn)

AI IMAGE PROMPTS
=================
File: references/image-prompts.md

Comprehensive library of 20+ prompts for generating professional marketing visuals using:
- Midjourney
- DALL-E
- Ideogram
- Flux

Categories:
1. Product Mockups (3 prompts) - Device mockups showing your app/website
2. Social Media Headers (3 prompts) - LinkedIn, Twitter, Instagram
3. Blog/Article Hero Images (4 prompts) - Blog post headers
4. Ad Creatives (4 prompts) - Eye-catching ads for various platforms
5. Presentation Backgrounds (4 prompts) - Slide backgrounds
6. Abstract/Conceptual (8 prompts) - AI, growth, innovation, security, etc.

Each prompt includes:
- Full prompt text ready to copy/paste
- Best tool recommendation
- Aspect ratio
- Style notes
- Variation suggestions

QUICK START
===========
1. Open a template in your browser
2. Edit the text to your liking
3. Take a screenshot (recommended: 1080px width)
4. Post to social media

For AI visuals:
1. Open image-prompts.md
2. Copy a prompt for your use case
3. Paste into Midjourney, DALL-E, Ideogram, or Flux
4. Generate and use the image

All templates and prompts use the brand color #2ABFAB for consistency.
